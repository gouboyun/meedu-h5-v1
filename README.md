### 快速上手

安装依赖：

```
npm install
```

修改配置：

```
cp .env.example .env
```

将 .env 文件中的 `VUE_APP_URL` 改为您的 meedu 网站地址（后面不要加 `/`）

编译：

```
npm run build
```

编译好，将会生成 `dist` 目录。将 `dist` 目录下的文件部署到新的站点。
